# Учёт клиентов для малого бизнеса (CRM) 📊

Добро пожаловать в репозиторий проекта **CRM для малого бизнеса**!  
Это простое, но функциональное приложение, которое поможет небольшим компаниям и индивидуальным предпринимателям вести базу клиентов, фиксировать историю взаимодействий и ничего не забывать.

## О проекте ❓

Данная CRM-система создана для автоматизации рутинных задач менеджеров по продажам и администраторов. Основная цель — собрать в одном месте всех клиентов, контактную информацию и историю общения (звонки, встречи), чтобы повысить эффективность работы и лояльность заказчиков.

### Основные возможности:
- **Управление клиентами**: хранение полной информации о компании и контактных лицах.
- **История событий**: запись всех взаимодействий (звонок, встреча, email) с датой и результатом.
- **Умный поиск и фильтры**: быстрый поиск нужного клиента или события.
- **Отчёты по активности**: анализ того, как часто менеджеры общаются с клиентами.

## Установка и запуск 🚀

Чтобы развернуть проект локально, выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ваш-аккаунт/crm-small-business.git
    ```
2.  **Перейдите в папку проекта:**
    ```bash
    cd crm-small-business
    ```
3.  **Установите зависимости** (пример для Node.js):
    ```bash
    npm install
    ```
4.  **Запустите приложение:**
    ```bash
    npm run start
    ```

После запуска приложение будет доступно по адресу `http://localhost:3000`.

## Сущности базы данных 🗂️

Проект построен на трёх основных сущностях:

*   **Клиент** — информация о компании-клиенте (название, адрес, ИНН).
*   **Контакт** — данные конкретного человека (ФИО, телефон, email, должность). Связан с клиентом.
*   **Событие** — запись о взаимодействии (тип: звонок/встреча, дата, время, краткий результат).

## Роли пользователей 👥

В системе предусмотрено два уровня доступа:

| Роль | Права |
| :--- | :--- |
| **Менеджер** | Просмотр клиентов, добавление событий, редактирование контактов. |
| **Администратор** | Полный доступ: управление клиентами, контактами, пользователями и просмотр отчётов. |
